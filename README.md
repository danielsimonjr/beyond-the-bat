# Beyond the Bat: Dissolving the Explanatory Gap Through Meaningful Integration

**Daniel Simon Jr.** | Independent Researcher | 2026

## Overview

This paper proposes the **Recursive Social Prediction (RSP)** model of consciousness — a three-level architecture in which phenomenal consciousness arises from the brain's integration of biological core affect with learned conceptual categories through recursive self-modeling.

The paper defends a **strong dissolution** of the hard problem: phenomenology IS recursive self-modeling, not a separate property that could be absent from it.

## The RSP Architecture

| Level | Name | Timescale | Function |
|-------|------|-----------|----------|
| 1 | Core Affect | ~100ms | Interoceptive monitoring → valence + arousal |
| 2 | Core Schemas | ~1-10s | Body, spatial, attentional, affective-homeostatic, self schemas |
| 3a | Conceptual Categorization | ~10s-min | Emotion categories, semantic knowledge (culturally learned) |
| 3b | Recursive Self-Modeling | ~10s-min | Strange loop: social prediction redirected at self → consciousness |
